## 题目链接

[反转字符串中的单词](https://leetcode.cn/problems/reverse-words-in-a-string/)

## 题目描述

给你一个字符串 `s`，请你反转字符串中 **单词** 的顺序。

> **单词定义：** 由非空格字符组成的字符串
> **分隔规则：** `s` 中使用至少一个空格将字符串中的 **单词** 分隔开
> **返回要求：** 单词顺序颠倒且单词之间用单个空格连接的结果字符串

> **注意事项：**
> - 输入字符串 `s` 中可能会存在前导空格、尾随空格或者单词间的多个空格
> - 返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格

### 示例

**示例 1：**
- **输入：** `s = "the sky is blue"`
- **输出：** `"blue is sky the"`

**示例 2：**
- **输入：** `s = "  hello world  "`
- **输出：** `"world hello"`
- **解释：** 反转后的字符串中不能存在前导空格和尾随空格

**示例 3：**
- **输入：** `s = "a good   example"`
- **输出：** `"example good a"`
- **解释：** 如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个

### 约束条件

- `1 <= s.length <= 10⁴`
- `s` 包含英文大小写字母、数字和空格 `' '`
- `s` 中 **至少存在一个** 单词

## 解题思路

### 采用原地修改+读写双指针（快慢指针）+头尾双指针的方法解决
需要将字符串的头部空格和尾部空格过滤，同时保持单词间的空格只有一个。读写双指针`read`和 `write`初始化为0（都从字符的开头开始），当`read`指向非空格字符时，需要循环将`read`指向的字符复制到`write`指向的位置，同时`read`和`write`都右移，直到`read`指向空格字符，再次遇到非空格的时候先将单词间的空格添加上再进行下一个单词的循环写入，最终形成一个新字符串（`s`的头部和尾巴都没有空格，但是单词间存在唯一空格）。

####

## 步骤

1. 构造函数`reverse`，用于反转字符串(也可以认为是反转单词)
2. 定义一个`space`变量，初始为 `1`，表示还没有遇到过任何单词，因此下一个单词前无需添加空格
3. 定义读指针`read`，用于遍历字符串、写指针`write`，用于写入新字符串
    - 开始利用读指针`read`遍历字符串
    - 如果`read`指向空格字符，直接read++（直接过滤掉多余空格）
    - 反之，则需要进行写入操作
        - 如果`space`为0且`write`大于0，先将单词间的空格添加上（也就是让`write`指向的位置写入空格 --- `s[write++] = ' '`），说明进入前`read`指向的是新单词，而`write`指向的是前一个单词的后一个位置
        - 循环将`read`指向的字符复制到`write`指向的位置，同时`read`和`write`都右移，直到`read`指向空格字符
        - 将`space`设置为0，说明当前单词已经写入完成，此时`read`和`write`都指向的是当前单词的后一个位置，退出当前循环
    - 重新进入循环，继续遍历处理
4. 循环结束，字符串反转完成(`s`就是反转后的字符串)

## 代码实现

```c
#include <string.h>

void reverse(char* s, int left, int right) {
    while (left < right) {
        char temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        left++;
        right--;
    }
}

char* reverseWords(char* s) {
    int len = strlen(s);
    int write = 0;
    int space = 1;
    for (int read = 0; read < len; read++) {
        if (s[read] != ' ') {
            if (!space && write > 0) 
                s[write++] = ' ';
            while (read < len && s[read] != ' ') 
                s[write++] = s[read++];
            space = 0;
        }
    }
    s[write] = '\0';

    reverse(s, 0, write - 1);
    len = strlen(s) + 1;
    int start = 0;
    for (int i = 0; i < len; i++) {
        if (s[i] == ' ' || s[i] == '\0') {
            reverse(s, start, i - 1);
            start = i + 1;
        }
    }
    return s;
}
```

***

### 时间复杂度
 
`O(n)` 其中 `n` 是字符串的长度。

### 空间复杂度

`O(1)` 只需要常数空间。
