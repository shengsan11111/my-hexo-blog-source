## 题目链接
[快乐数](https://leetcode.cn/problems/happy-number/)

## 题目描述

编写一个算法来判断一个数 `n` 是不是**快乐数**。

> **快乐数定义：**
> 对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。
> 然后重复这个过程直到：
> - 这个数变为 **1**（是快乐数）
> - 进入**无限循环**但始终变不到 1（不是快乐数）

### 示例

**示例 1：**
- **输入：** `n = 19`
- **输出：** `true`
- **解释：**
  ```
  1² + 9² = 82
  8² + 2² = 68
  6² + 8² = 100
  1² + 0² + 0² = 1
  ```

**示例 2：**
- **输入：** `n = 2`
- **输出：** `false`

### 提示
- `1 <= n <= 2³¹ - 1`

## 解题思路
### 采用哈希表作为标记数组的方法解决

创建一个哈希表（用来记录已经出现过的平方和），将每次获取到的平方和作为哈希表的下标来访问哈希表，先判断当前的平方和是否为1，若为1，则返回true，反之判断哈希表对应的值是否为false，若为true，则返回false，因为如果哈希表对应的值为true，说明该平方和之前已经出现过，说明肯定陷入了死循环，如果哈希表对应的值为false，则说明该平方和之前没有出现过，说明还可以继续循环，将该值置为true，继续进行下一次循环。

##  步骤

1. 构造一个函数getNextNumber，用于计算下一个平方和
2. 构造isHappy函数，用于判断一个数是否为快乐数
3. 在isHappy函数中，若sum为1，则返回true，若hash[sum]为true，则返回false，若hash[sum]为false，则将该hash[sum]置为true，继续进行下一次循环。

## 代码实现
```c
int getNextNumber(int x) {
    int sum = 0;
    while (x > 0) {
        int d = x % 10;
        sum += d * d;
        x /= 10;
    }
    return sum;
}

bool isHappy(int n) {
    bool hash[1000] = {false};
    while (n != 1) {
        int sum = getNextNumber(n);
        if (sum == 1) 
            return true;
        if (hash[sum]) 
            return false;
        hash[sum] = true;
        n = sum;
    }
    return true;
}
```

### 时间复杂度 O(logn)
### 空间复杂度 O(1)
