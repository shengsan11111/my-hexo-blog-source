## 题目链接：[四数之和](https://leetcode.cn/problems/4sum/)

## 题目描述

给你四个整数数组 `nums1`、`nums2`、`nums3` 和 `nums4`，数组长度都是 `n`，请你计算有多少个**元组** `(i, j, k, l)` 能满足：

- `0 <= i, j, k, l < n`
- `nums1[i] + nums2[j] + nums3[k] + nums4[l] == 0`

### 示例

**示例 1：**
- **输入：** 
  ```
  nums1 = [1,2], nums2 = [-2,-1], nums3 = [-1,2], nums4 = [0,2]
  ```
- **输出：** `2`
- **解释：**
  两个元组如下：
  1. `(0, 0, 0, 1)` → `nums1[0] + nums2[0] + nums3[0] + nums4[1] = 1 + (-2) + (-1) + 2 = 0`
  2. `(1, 1, 0, 0)` → `nums1[1] + nums2[1] + nums3[0] + nums4[0] = 2 + (-1) + (-1) + 0 = 0`

**示例 2：**
- **输入：** 
  ```
  nums1 = [0], nums2 = [0], nums3 = [0], nums4 = [0]
  ```
- **输出：** `1`

### 约束条件

- `n == nums1.length == nums2.length == nums3.length == nums4.length`
- `1 <= n <= 200`
- `-2²⁸ <= nums1[i], nums2[i], nums3[i], nums4[i] <= 2²⁸`

## 解题思路
### 采用分组+二分查找的方法解决
#### 数学公式：nums1[i]+nums2[j]+nums3[k]+nums4[l]=0 ----> nums1[i]+nums2[j]=-(nums3[k]+nums4[l])
先将四个数组分成两大组（nums1和nums2，nums3和nums4），先计算nums1和nums2的所有的可能和(双重循环)存入sums数组中，再对sums数组进行升序排序，最后以相同的方法（双重循环）计算nums3和nums4的所有可能和，取nums3和nums4的所有可能和的相反数，再用二分查找法统计满足条件的组合数（因为sums数组中可能存在多个对应相反数值，所以一个函数查找相反数的起始位置，一个函数查找相反数的结束位置）


##  步骤

1. 构造cmp函数，用于快速排序,设置为升序排序规则
2. 构造target_start函数，用于查找sums数组中是否存在满足target的起始位置
3. 构造target_end函数，用于查找sums数组中是否存在满足target的结束位置(不包括target)
4. 先将nums1和nums2的所有可能和(双重循环)存入sums数组中，再对sums数组进行升序排序（利用快排）
5. 最后以相同的方法（双重循环）计算nums3和nums4的所有可能和，取nums3和nums4的所有可能和的相反数target，通过函数target_start查找sums中target的起始位置存入low，通过函数target_end查找sums中target的结束位置存入high，high-low为当前target在sums数组中出现的次数
6. 统计不同的target值，每个target值的组合数为high-low，将所有target值的组合数累加起来，就是最终的结果
7. 返回count

### 注意：利用双指针的方法来查找sums数组中是否存在满足条件的组合数（会在力扣当中运行超时），如果想使用双指针法查找满足条件的组合数，可以去做力扣移除元素的问题。

## 代码实现

```c
#include <stdlib.h>

// 比较函数，用于快速排序,设置为升序排序规则
int cmp(const void* a, const void* b) {
    return *(int*)a - *(int*)b;
}

// 采用的是左闭右开区间，也有左闭右闭区间
// 二分查找：寻找目标值的起始位置
int target_start(int* arr, int size, int target) {
    int left = 0, right = size;
    while (left < right) {
        int mid = (left + right) / 2;
        if (arr[mid] < target)
            left = mid + 1;
        else
            right = mid;
    }
    return left;
}

// 二分查找：寻找目标值的下一个位置（不包括目标值）
int target_end(int* arr, int size, int target) {
    int left = 0, right = size;
    while (left < right) {
        int mid = (left + right) / 2;
        if (arr[mid] <= target)
            left = mid + 1;
        else
            right = mid;
    }
    return left;
}

int fourSumCount(int* nums1, int nums1Size, int* nums2, int nums2Size,
                 int* nums3, int nums3Size, int* nums4, int nums4Size) {
    int n = nums1Size;
    
    int* sums = (int*)malloc(n * n * sizeof(int));
    int idx = 0;
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            sums[idx++] = nums1[i] + nums2[j];
        }
    }
    qsort(sums, n * n, sizeof(int), cmp);
    
    int count = 0;
    for (int k = 0; k < n; ++k) {
        for (int l = 0; l < n; ++l) {
            int target = -(nums3[k] + nums4[l]);
            int start = target_start(sums, n * n, target);
            int end = target_end(sums, n * n, target);
            count += end - start;
        }
    }
    
    free(sums);
    return count;
}
```

### 时间复杂度

O(n²logn) 排序时间为 O(n²logn)，二分查找时间为 O(n²)，所以取大的 O(n²logn)

### 空间复杂度

O(n²) 最坏情况下需要存储 n² 个元素
