## 题目链接

[赎金信](https://leetcode.cn/problems/ransom-note/description/)

## 题目描述

给你两个字符串：`ransomNote` 和 `magazine`，判断 `ransomNote` 能不能由 `magazine` 里面的字符构成。

> **规则：**
> - 如果可以，返回 `true`；否则返回 `false`
> - `magazine` 中的每个字符只能在 `ransomNote` 中使用一次

### 示例

**示例 1：**
- **输入：** `ransomNote = "a"`, `magazine = "b"`
- **输出：** `false`

**示例 2：**
- **输入：** `ransomNote = "aa"`, `magazine = "ab"`
- **输出：** `false`

**示例 3：**
- **输入：** `ransomNote = "aa"`, `magazine = "aab"`
- **输出：** `true`

### 约束条件

- `1 <= ransomNote.length, magazine.length <= 10⁵`
- `ransomNote` 和 `magazine` 由小写英文字母组成

---

## 解题思路

### 采用双循环的方式来解决(外层循环遍历ransomNote，内层循环遍历magazine，查找未使用的匹配字符)
定义一个标记数组used用来存放magazine中每个字符是否被使用过。外层循环遍历ransomNote，内层循环遍历magazine，若magazine中存在未使用的匹配字符，则标记为已使用true。若不存在，则返回false。


## 步骤

1. 初始化标记数组used，长度为magazine的长度，将所有元素初始化为false（表示一开始每个字符未被使用过）
2. 外层遍历ransomNote，内层遍历magazine，依次查找未使用的匹配字符。
   - 若magazine中存在未使用的匹配字符，则标记为已使用true，跳出内层循环。
   - 若不存在，则返回false
3. 返回true/false（若ransomNote中所有字符都能由magazine中的字符构成，则返回true）

---

## 代码实现

```c
bool canConstruct(char * ransomNote, char * magazine) {
    int len1 = strlen(ransomNote);
    int len2 = strlen(magazine);
    bool used[len2];
    //初始化标记数组used，将所有元素初始化为false（表示一开始每个字符未被使用过）
    for (int i = 0; i < len2; i++) {
        used[i] = false;
    }
    for (int i = 0; i < len1; i++) {
        bool found = false;
        //遍历magazine，查找未使用的匹配字符
        for (int j = 0; j < len2; j++) {
            //若magazine中存在未使用的匹配字符，则标记为已使用true，跳出内层循环。
            if (used[j]==false && magazine[j] == ransomNote[i]) {
                used[j] = true;
                found = true;
                break;
            }
        }
        //若magazine中不存在未使用的匹配字符，则返回false
        if (!found) {
            return false;
        }
    }
    return true;
}
```

---

### 时间复杂度

O(n*m) 最坏情况下需要遍历外层循环n次，内层循环m次，时间复杂度为 O(n*m)

### 空间复杂度

O(m) 需要定义的布尔数组标记已使用的字符，也就是magazine的长度，空间复杂度为 O(m)

## 优化解题思路
### 采用哈希表的方式来解决
定义一个哈希表来存储magazine中每个字符的出现次数。遍历ransomNote,消耗哈希表中对应的字符次数。若哈希表中对应的字符次数小于0，则返回false。

---

## 步骤

1. 初始化哈希表hash，长度为26，将所有元素初始化为0(表示每个字符出现次数为0)
2. 遍历magazine，将每个字符的出现次数存储到哈希表中(利用字符偏移作为下标来存储)
    -例如：字符'a'的下标为0（偏移量为0）
3. 遍历ransomNote，消耗哈希表中对应的字符次数
    - 若哈希表中对应的字符次数等于0，则返回false
    - 若哈希表中对应的字符次数大于0，则对应字符次数减1
4. 返回true/false（若ransomNote中所有字符都能由magazine中的字符构成，则返回true）

## 代码实现
```c
bool canConstruct(char * ransomNote, char * magazine) {
    //初始化26个字母的出现次数为0
    int hash[26] = {0};
    //遍历magazine，将每个字符的出现次数存储到哈希表中
    while (*magazine) {
        hash[*magazine - 'a']++;
        magazine++;
    }
    //遍历ransomNote，消耗哈希表中对应的字符次数
    while (*ransomNote) {
        int i = *ransomNote - 'a';
        //若哈希表中对应的字符次数等于0，说明magazine中字符次数不足，返回false
        if(hash[i] == 0){
            return false;
        }
        hash[i]--;
        ransomNote++;
    }
    
    return true;
}
```

### 时间复杂度

O(n+m) 最坏情况下需要遍历赋值标记数组used需要n次，遍历ransomNote需要m次，所以时间复杂度为 O(n+m)

### 空间复杂度

O(1) 需要定义的哈希表hash，空间复杂度为 O(1)，大小为26个
