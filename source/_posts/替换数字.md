## 题目描述

给定一个字符串 `s`，它包含小写字母和数字字符，请编写一个函数，将字符串中的字母字符保持不变，而将每个数字字符替换为 `number`。

**示例：** 对于输入字符串 "a1b2c3"，函数应该将其转换为 "anumberbnumbercnumber"。

## 输入描述

输入一个字符串 `s`，`s` 仅包含小写字母和数字字符。

## 输出描述

打印一个新的字符串，其中每个数字字符都被替换为了 `number`。

### 输入示例

```
a1b2c3
```

### 输出示例

```
anumberbnumbercnumber
```

### 提示信息

- **数据范围：** `1 <= s.length < 10000`

***

## 解题思路

### 采用原地修改+双指针的方法解决
准备一个足够大的数组 `s`存储替换后的字符串，同时s是输入字符串的存储数组。定义一个首尾指针 `oldendindex` 和 `newendindex`，分别指向旧字符串的末尾和新字符串的末尾。`oldendindex`往前遍历判断字符类型，`newendindex`往前遍历通过字符类型来判断是直接用旧字符还是替换为`number`，直到`oldendindex`小于0。


## 步骤

1. 设置一个足够存储替换后的字符串的数组 `s`，同时s是输入字符串的存储数组。
2. 定义一个首尾指针 `oldendindex` 和 `newendindex`，分别指向旧字符串的末尾和新字符串的末尾。
   - 遍历旧字符串s，从后往前遍历
   - 如果当前`oldendindex`指向的是非数字字符，直接将该字符复制到新字符串的末尾，`oldendindex` 左移一位和 `newendindex` 左移一位
   - 如果当前`oldendindex`指向的是数字字符，将`newendindex`指向的位置逐步移动一位，逐步替换为`number`，需要移动6位，最终整体替换为`number`，`oldendindex` 左移1位
   - 直到`oldendindex`小于0则停止遍历
3. 循环结束，字符串反转完成(`s`就是反转后的字符串)

***

## 代码实现

```c
#include <stdio.h>
#include <string.h>

int main() {
    char s[60010];
    while (scanf("%s", s) == 1) {
        int len = strlen(s);
        int count = 0;
        for (int i = 0; i < len; ++i) {
            if (s[i] >= '0' && s[i] <= '9') 
                count++;
        }
        int newLen = len + count * 5;
        int oldendindex = len - 1;
        int newendindex = newLen - 1;
        s[newLen] = '\0';
        while (oldendindex >= 0) {
            if (s[oldendindex] >= '0' && s[oldendindex] <= '9') { 
                s[newendindex--] = 'r';
                s[newendindex--] = 'e';
                s[newendindex--] = 'b';
                s[newendindex--] = 'm';
                s[newendindex--] = 'u';
                s[newendindex--] = 'n';
            } 
            else {
                s[newendindex--] = s[oldendindex];
            }
            --oldendindex;
        }
        printf("%s\n", s);
    }
    return 0;
}
```

***

### 时间复杂度

`O(n)` 每个字符只访问一次。

### 空间复杂度

`O(1)` 只需要常数空间。
