## 题目链接

[反转字符串](https://leetcode.cn/problems/reverse-string/)

## 题目描述

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 `s` 的形式给出。

> **要求：**
> - 不要给另外的数组分配额外的空间
> - 必须**原地**修改输入数组
> - 使用 **O(1)** 的额外空间解决这一问题

### 示例

**示例 1：**
- **输入：** `s = ["h","e","l","l","o"]`
- **输出：** `["o","l","l","e","h"]`

**示例 2：**
- **输入：** `s = ["H","a","n","n","a","h"]`
- **输出：** `["h","a","n","n","a","H"]`

### 约束条件

- `1 <= s.length <= 10⁵`
- `s[i]` 都是 ASCII 码表中的可打印字符

***

## 解题思路

### 采用原地修改+双指针的方法解决
利用双指针`left`和 `right`，分别指向字符串的开头和结尾，交换它们指向的字符，同时向中间移动。

####

## 步骤

1. 初始化左右指针`left`和 `right`，分别指向字符串的开头(`0`)和结尾(`sSize-1`)
2. 当 `left < right` 时，交换 `s[left]` 和 `s[right]`
3. 左指针 `left` 右移，右指针 `right` 左移
4. 循环结束，字符串反转完成(`s`就是反转后的字符串)

###

***

## 代码实现

```c
void reverseString(char* s, int sSize) {
    int left = 0,right = sSize-1;
    while(left < right){
        char temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        left++;
        right--;
    }
}
```

***

### 时间复杂度

`O(n)` 其中 `n` 是字符串的长度。

### 空间复杂度

`O(1)` 只需要常数空间。
