## 题目描述

字符串的**右旋转**操作是把字符串尾部的若干个字符转移到字符串的前面。给定一个字符串 `s` 和一个正整数 `k`，请编写一个函数，将字符串中的**后面 k 个字符移到字符串的前面**，实现字符串的右旋转操作。

**示例：** 对于输入字符串 "abcdefg" 和整数 `2`，函数应该将其转换为 "fgabcde"。

## 输入描述

输入共包含两行：

- 第一行为一个正整数 `k`，代表右旋转的位数
- 第二行为字符串 `s`，代表需要旋转的字符串

## 输出描述

输出共一行，为进行了右旋转操作后的字符串

### 输入示例

```
2
abcdefg
```

### 输出示例

```
fgabcde
```

### 提示信息

- **数据范围：**
  - `1 <= k < 10000`
  - `1 <= s.length < 10000`

***

## 解题思路

### 采用 原地修改 + 双指针 的方法解决
准备一个足够大的字符数组存储输入字符串`s`，同时作为输出字符串的存储数组。利用双指针`left`和 `right`，分别指向区间字符串的开头和结尾，交换它们指向的字符，同时向区间中间移动，先完全反转整个字符串，再区间反转前`k`个字符，再区间反转剩余字符。

## 步骤

1. 设置一个足够存储替换后的字符串的数组 `s`，同时s是输入字符串的存储数组。
2. 构造函数 `reverse`，用于反转区间字符串
    - 当 `left < right` 时，交换 `s[left]` 和 `s[right]`
    - 左指针 `left` 右移，右指针 `right` 左移
    - 循环结束，区间字符串反转完成
3. 执行
    - 先完全反转整个字符串
    - 再区间反转前 `k` 个字符
    - 最后区间反转剩余字符（从 `k` 开始到字符串末尾）

***

## 代码实现

```c
#include <stdio.h>
#include <string.h>

void reverse(char* s, int left, int right) {
    while (left < right) {
        char temp = s[left];
        s[left] = s[right];
        s[right] = temp;
        left++;
        right--;
    }
}

int main() {
    int k;
    char s[10010];
    scanf("%d", &k);
    scanf("%s", s);
    int len = strlen(s);
    k %= len;
    reverse(s, 0, len - 1);
    reverse(s, 0, k - 1);
    reverse(s, k, len - 1);
    printf("%s\n", s);
    return 0;
}
```

***

### 时间复杂度

`O(n)` 每个字符只访问一次。

### 空间复杂度

`O(1)` 只需要常数空间。
