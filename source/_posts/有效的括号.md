## 题目链接：[有效的括号](https://leetcode.cn/problems/valid-parentheses/)

## 题目描述

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s`，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

## 示例

**示例 1：**

```
输入：s = "()"
输出：true
```

**示例 2：**

```
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```
输入：s = "(]"
输出：false
```

**示例 4：**

```
输入：s = "([])"
输出：true
```

**示例 5：**

```
输入：s = "([)]"
输出：false
```

## 约束条件

- `1 <= s.length <= 10^4`
- `s` 仅由括号 `'()[]{}'` 组成

---

## 解题思路
### 采用栈解决括号匹配问题
### 为什么采用栈 - 可能是为了比较相邻的两个字符
定义一个栈，用于存储左括号，遍历字符串，当前字符如果是规定的左括号，直接入栈；如果是其他字符，判断栈顶字符是否为当前栈顶左括号对应的右括号，满足则出栈，反之返回false。

##  步骤

1. 初始化一个栈，用于存储左括号，`length` 为栈的长度。
2. 遍历字符串：
    - 如果是规定的左括号，直接入栈。
    - 如果是右括号，判断栈顶字符是否为当前栈顶左括号对应的右括号
        - 如果是同一组左右括号，出栈（`length--`）。
        - 如果不是同一组左右括号，返回false。
3. 如果遍历完成后，栈为空（说明所有左括号都有对应的右括号匹配），返回true；否则返回false。
---

## 代码实现

```c
typedef struct{
    char *s;
    int length;
} mystack;
bool isValid(char *s) {
    int len = strlen(s);
    mystack* Mystack = (mystack *)malloc(sizeof(mystack));
    Mystack->length = 0;
    Mystack->s = (char *)malloc(sizeof(char) * len);
    if (len % 2 == 1) 
        return false;
    for (int i = 0; i < len; i++) {
        if (s[i] == '(' || s[i] == '{' || s[i] == '[') {
            Mystack->s[Mystack->length] = s[i];
            Mystack->length++;
        } 
        else {
            if (Mystack->length == 0) 
                return false;
            char c = Mystack->s[Mystack->length-1];
            Mystack->length--;
            if ((s[i] == ')' && c != '(') ||
                (s[i] == '}' && c != '{') ||
                (s[i] == ']' && c != '['))
                return false;
        }
    }
    return Mystack->length == 0;
}
```

---

### 时间复杂度

`O(n)` 遍历字符串一次，每个字符最多入栈一次出栈一次，所以时间复杂度为 `O(n)`。

### 空间复杂度

`O(n)` 栈最多存放 n 个字符（全部为左括号时），所以空间复杂度为 `O(n)`。
