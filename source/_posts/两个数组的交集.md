## 题意
给定两个数组，编写一个函数来计算它们的交集。

**提示：**
- 1 <= nums1.length, nums2.length <= 1000
- 0 <= nums1[i], nums2[i] <= 1000

## 思路
1、创建两个数组，分别为array和ret，array存放标记过的元素，ret存放交集元素
2、遍历nums1数组，将元素放入array对应的下标位置，例如：nums1[1] = 1000，array[1000] = 1
3、遍历nums2数组，判断array[nums2[i]]是否为1，若为1，则将nums2[i]放入ret数组中，每次放一次都要让ret数组的下标（index）加1，将array[nums2[i]]置为0（表示已访问）
4、返回ret数组，index为交集元素的个数

代码实现：
int* intersection(int* nums1, int nums1Size, int* nums2, int nums2Size,int* returnSize) {
    int array[1001] = {0};
    //交集是取较小长度，并集才是取较大长度
    int* ret = (int*)malloc(sizeof(int) *(nums1Size < nums2Size ? nums1Size : nums2Size));
    // 初始化返回ret数组的大小为0
    int index = 0;
    for (int i = 0; i < nums1Size; i++) {
        array[nums1[i]] = 1;
    }
    for (int i = 0; i < nums2Size; i++) {
        //判断array[nums2[i]]是否为1
        if (array[nums2[i]] == 1) {
            ret[index] = nums2[i];
            // 避免重复添加，所以将array[nums2[i]]改回为0
            array[nums2[i]] = 0;
            // 每次放一次都要让ret数组的下标（index）加1
            index++;
        }
    }
    *returnSize = index;
    return ret;
}
